# Financial Data Analytics with Python, SQL & Power BI

## 📊 Project Overview
This project focuses on analyzing banking data to extract meaningful insights using Python, SQL, and Power BI. It demonstrates a complete data analytics workflow including data cleaning, exploration, and visualization.

The goal of this project is to understand customer behavior, transaction patterns, and financial trends in the banking domain.

---

## 🚀 Tools & Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
- Jupyter Notebook
